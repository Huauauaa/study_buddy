# study_buddy

## prepare

### create virtual env

`virturalenv [envname]`

### active virtual env

- win: `[envname]\Scripts\activate`
- mac: `source [envname]/bin/activate`

### freeze labs

`pip freeze > requirements.txt`

### install labs

`pip install -r requirements.txt`

## create project

`django-admin startproject study_buddy`

## run project

`python manage.py runserver`
